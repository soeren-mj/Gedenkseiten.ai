# Kondolenzbuch Preset Bilder

Diese Bilder werden für die Deckblatt-Vorlagen des Kondolenzbuchs benötigt.

## Benötigte Bilder

| Dateiname | Beschreibung | Empfohlene Textfarbe |
|-----------|--------------|---------------------|
| `kerzen.jpg` | Kerzen/Kerzenlicht | Weiß |
| `blumen.jpg` | Blumen | Weiß |
| `natur.jpg` | Natur-Landschaft | Weiß |
| `himmel.jpg` | Himmel/Wolken | Weiß |
| `ozean.jpg` | Ozean/Meer | Weiß |
| `wald.jpg` | Wald | Weiß |

## Anforderungen

- **Format:** JPG
- **Empfohlene Größe:** 800x1067px (3:4 Aspect Ratio)
- **Qualität:** Mittlere Kompression für Web
- **Stil:** Ruhige, würdevolle Motive passend für Kondolenzbücher

## Hinweise

- Bilder sollten lizenzfrei oder mit entsprechender Lizenz sein
- Dunkle oder mittlere Töne funktionieren besser für weißen Text
- Falls keine echten Bilder vorhanden, werden Platzhalter angezeigt
